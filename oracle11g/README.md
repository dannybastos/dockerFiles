# Oracle 11g XE

### This is a sample docker-compose for oracle-11g-xe

## System requirements
- Docker
- Docker-compose

## How to run
```
docker-compose up
```

This container exposes the follow ports:
- 1521 - db
- 8080 - apex

Default username and password
- SYS, SYSTEM : oracle

Apex web console :
- workspace: internal
- username: admin
- password: oracle
